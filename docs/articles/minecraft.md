# Minecraft

# 概述

用于记录服务器搭建、模组生态与资源制作，偏实践指南。

# 服务器与模组

- 平台：Paper、Purpur、Fabric、Forge
- 选择策略：性能优先用 Paper，模组生态选 Fabric

# 数据包与资源包

- 数据包：函数、进度、战利品表
- 资源包：模型与材质，字体与语言文件

# 常用工具

- WorldEdit、FAWE
- NBT Explorer、BlockBench

# 指令与自动化

- 基础：give、tp、scoreboard
- 自动化：函数链与定时执行

# 性能优化要点

- 限制实体堆叠与刷怪
- 视距、区块加载与 I/O
- 监控：spark、timings
